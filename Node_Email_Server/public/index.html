Node.js Email Server

Send confirmation emails from a simple HTTP server

This project demonstrates Node basics, static file serving, request logging, and email delivery through a single server.

Send Email

How it works

  • Static pages are served with Node's built-in `http` module.
  • Requests are logged with timestamps in the terminal.
  • Form submissions are sent to `/send` using `fetch`.
  • Emails are delivered using `nodemailer`.
